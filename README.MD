
# ImageViewer

------------

### 打开图片
	- 文件过滤器
	- 自动读取同文件夹下其它图片缓存起来

### 查看图片：
	- 放大
	- 缩小
	- 上一张
	- 下一张
	- 与窗口同宽显示
	- 显示原图（100%）
